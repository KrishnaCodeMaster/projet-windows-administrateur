# Projet Windows Administrator - Service pour gestion des besooins IT d'une entreprise
Le but de ce projet été de répondre aux besoins IT d'une entreprise en utilisants windows server management.

### Ce que j'ai appris
J'ai appris à manipuler des Virtual Machine. A créer des Active Directory pour mettre les différents machines dans le même domaine. Appliquer des GPO, Créer une connexion VPN entre les différents sites (Virtual Machine). Créer des réplication de fichiers sur les sites qui se font automatiquement
